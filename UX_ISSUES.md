# 🌱 巡園紀錄（UX_ISSUES.md）

VERA 平常逛莊園時隨手記的小卡頓，不用停下來修、不用寫技術細節。累積 5～10 筆再一次交給工程端收斂。

這是一個「問題停車場」——先把問題停在這裡，避免「今天想到、改一半、忘記原本為什麼改」。
不是改功能，只是先把發現的東西記下來。

**格式（每筆固定五項）：**

```
### 📍 位置
（在哪個分頁/哪張卡）

問題：
（客觀描述哪裡怪怪的/卡住了）

第一次使用者可能誤會：
（猜猜看新來的人會怎麼誤解這裡，沒有頭緒也可以留空）

方向：
（有想法就寫，沒有也沒關係）

狀態：
待修 / 已修（日期）
```
